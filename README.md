# Guess the Number Game 

This is a simple number guessing game made by using python.

## How the game works
- The computer generates a random number between 1 and 10.
- The player has 3 attempts to guess the number.
- If the guess is correct, the player wins.
- If all attempts are wrong, the game ends and the correct number is shown.

## How to run the game
- Make sure Python is installed.
- Download or clone this repository
- Run the file using:
   python Game.py

## Features
- Random number generation
- Limited attempts (3 tries)
- Beginner-friendly Python code
